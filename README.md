# 🚗 Car Class Project - C++

## 📖 About The Project
This is a simple C++ project built using basic Object-Oriented Programming (OOP) concepts.

The project demonstrates how to:
- Create a class
- Define attributes (data members)
- Implement member functions
- Create objects from a class
- Access and display object data

## 🧠 Concepts Used
- Classes and Objects
- Public data members
- Member functions
- Basic structure of OOP in C++
